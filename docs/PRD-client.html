1. ภาพรวมโครงการ

ปัจจุบันเว็บของคลินิกแยกเป็น 3 ส่วนที่ไม่เชื่อมกัน ทำให้ดูไม่เป็นอันหนึ่งอันเดียว และดูแลยาก เราจะ รวมทุกอย่างให้เป็นเว็บเดียว ที่หน้าตาสวยสอดคล้องกันทั้งหมด

2. ปัญหาของเว็บเดิม

  • หน้าตาเก่า ไม่สอดคล้องกัน — แต่ละส่วนของเว็บดูคนละสไตล์ ทำให้แบรนด์ดูไม่เป็นมืออาชีพเท่าที่ควร
  • ใช้บนมือถือไม่สะดวก — เมนูและการเลื่อนหน้ายังไม่เหมาะกับมือถือ ทั้งที่ลูกค้าส่วนใหญ่เข้าจากมือถือ
  • Google ค้นเจอยากในบางหน้า — โครงสร้างเว็บบางส่วนทำให้ Google เข้าใจเนื้อหาได้ไม่ดี
  • เนื้อหาซ้ำซ้อนกันเยอะ — มีบทความและรีวิวจำนวนมากที่เนื้อหาทับกัน ทำให้ทั้งคนอ่านและ Google สับสน
  • โหลดช้าในบางหน้า — กระทบทั้งประสบการณ์ลูกค้าและอันดับบน Google

3. โครงสร้างเว็บใหม่

เราจะจัดเนื้อหาทั้งหมดให้เป็นระเบียบ ค้นง่าย โดยรวบรวมจากเว็บเดิมทั้ง 3 ส่วน ดังนี้

หน้าหลักของเว็บ

  • หน้าแรก — แนะนำคลินิก บริการเด่น และรีวิว
  • เกี่ยวกับเรา — เรื่องราวคลินิก ทีมแพทย์ และใบรับรอง
  • คำถามที่พบบ่อย
  • ติดต่อเรา — แผนที่ เบอร์โทร ไลน์ และฟอร์มจองคิว

4. สิ่งที่ลูกค้าจะได้รับ

  • เว็บไซต์ใหม่ที่สวย ทันสมัย ใช้งานง่ายทุกอุปกรณ์
  • ระบบให้คลินิก แก้ไขเนื้อหาเองได้ (เพิ่มบทความ รีวิว บริการ โดยไม่ต้องพึ่งโปรแกรมเมอร์)
  • ปุ่มจองคิว / ทักไลน์ / โทร ที่เห็นชัดและกดง่าย
  • ย้ายเนื้อหาเดิมทั้งหมดมาให้ครบ พร้อม รักษาอันดับ Google ที่มีอยู่ ไม่ให้หล่นหาย
  • เว็บที่ Google เข้าใจง่ายขึ้น = โอกาสติดอันดับและมีลูกค้าเข้ามามากขึ้น

5. สิ่งที่ต้องการจากลูกค้า

  1. กำหนดวันที่อยากเปิดเว็บ (ถ้ามี)
  2. โลโก้ สี และภาพลักษณ์แบรนด์ ที่ต้องการ (หรือให้เราออกแบบใหม่ให้)
  3. สิทธิ์เข้าถึงเว็บเดิมและบัญชีที่เกี่ยวข้อง (เพื่อย้ายข้อมูลและรักษาอันดับ Google)
  4. ช่องทางจองคิว/ติดต่อ ที่ใช้อยู่ (ไลน์ / เบอร์โทร / ระบบนัดหมาย)
  5. ผู้ที่จะดูแลเนื้อหาหลังเปิดเว็บ ฝั่งคลินิก
  6. เว็บดูดไขมัน (โดเมนแยก) — ยังต้องการใช้งานต่อหรือรวมเข้าเว็บใหม่?
